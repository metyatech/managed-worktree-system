# Security Policy

If you find a security issue in this repository, report it privately before public disclosure.

## Supported versions

Only the latest published major version is supported with security fixes.

## Reporting

- Open a private GitHub security advisory if the issue affects the published package or repository automation.
- If a security advisory is not available, contact the repository owner privately through GitHub before disclosure.

Please include:

- the affected `mwt` version
- the impacted command or workflow
- reproduction steps
- any proof-of-concept repository layout needed to reproduce the issue
